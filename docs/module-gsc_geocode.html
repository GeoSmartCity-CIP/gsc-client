(static) geocode(geonames) → {Promise.<gsc.Response>}

Get a list of matching coordinate entries for a user supplied name. The matches will include the matching name, a ranking and spatial coordinates according to the EPSG:4326 spatial reference system.

(static) reverseGeocode(lat, lon) → {Promise.<gsc.Response>}

Get a list of matching names for a user supplied coordinate pair in the EPSG:4326 spatial reference system. The returned list includes the formal name, the administrative area, the country and the distance from the user supplied coordinate as well as the actual spatial coordinates (in the same SRS) as the above method.
