Class: Route

gsc/routing.Route

- A class that represents a calculated route with a start and an end point

Constructor

new Route(name, startingPoint, endPoint, distance, segments)

Create a new instance of a Route object
Parameters:
Name Type Description
name String A descriptive name that the route will be saved under. Will also be used in the route list.
startingPoint ol.geom.Point The longitude/latitude pair that defines the start of the route
endPoint ol.geom.Point The longitude/latitude pair that defines the end point of the route
distance Number A geojson object that defines a route
segments GeoJSON.FeatureCollection A geojson FeatureCollection object that defines the segments of the route
