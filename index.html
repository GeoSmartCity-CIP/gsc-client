gsc.js

GeoSmartCity Client Side JavaScript Library API

gsc.js is a productivity library for rapid spatial web application development. The library builds on jQuery, OpenLayers 3 and invokes methods from an instance of the GeoSmartCity Hub

Visit project web site »

Get the library

gsc.js is being developed as a community effort on the social coding platform GitHub. It is released under a permissive Open Source EUPL v.1.1 licence and can be downloaded and used by anyone for any purpose.
